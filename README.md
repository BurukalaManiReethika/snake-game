# 🐍 Snake Game

A classic Snake Game built using HTML, CSS, and JavaScript.

## Features

- Keyboard controls
- Food generation
- Score tracking
- Collision detection
- Restart button

## Tech Stack

- HTML5 Canvas
- CSS3
- JavaScript

## How to Play

- ⬅️ Arrow Left
- ➡️ Arrow Right
- ⬆️ Arrow Up
- ⬇️ Arrow Down

Eat food to grow and increase your score.
